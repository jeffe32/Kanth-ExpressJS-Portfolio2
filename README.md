Overview

This project is a Personal Portfolio Website that was developed using Node.js, Express.js, also as well as EJS templating engine
also it shows my background, the projects that I have done, also my contact details in a well structured and repsonsive design

This site uses Express routes to navigate between the pages and also includes a shared header and also footer partials
to help maintain the design consistency accross all the views.
The application has a modern design implemented using Bootstrap 5.33 as well as font awesome icons.

Features
Built using ExpressJS and EJS templating engine
  - Home Page - Introduction and overview
  - About Me Page - Personal background and hobbies
  - Projects Page = Description of academic projects
  - Contact Me Page - Form for user input
Uses partials (header.ejs and footer.ejs) for reusable layout sections
Fully Bootstrap-based responsive design
Font Awesome icons for professional UI enhancement
Organized asset structure with public folders for images, videos, and CSS
Version control via GitHub with multiple commits showing development progress
Cloud deployment (Render)
